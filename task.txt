Design a URL shortener service thattakes in a valid URL and returns a shortened URL, redirecting the user to the previously provided URL.

Also, keep track of total visits/clicks on the URL.

Routes

POST /URL - Generates a new short URL and return s the shoirtened URL in the format example.com/random-id

GET /:id - Redirects the user to the original URL.

Get /URL/analytics/:id - Returns the clicks for the provided short id.
